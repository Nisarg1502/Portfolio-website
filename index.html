Language Known

HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications. It's the foundation of any website you see in a browser. HTML uses tags to structure content, defining elements like headings, paragraphs, images, links, and more. Browsers interpret these tags to display the content to users.

CSS

CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation of a document written in a markup language 1 like HTML. It controls the look and formatting of a webpage, allowing you to style things like colors, fonts, layout, and more. Essentially, CSS is what makes a webpage look good!

JavaScript

JavaScript is a versatile programming language primarily used to add interactivity and dynamic behavior to websites. It allows developers to create engaging user experiences, handle user input, manipulate web page content, and much more. JavaScript is also used in other environments like server-side scripting (Node.js) and mobile app development.

BootStrap

Bootstrap is a popular, free, and open-source CSS framework that makes it easier to build responsive, mobile-first websites. It provides ready-made CSS styles and JavaScript components for things like grids, typography, forms, buttons, navigation, and more. Essentially, Bootstrap gives you a solid foundation and speeds up web development by providing a consistent and customizable structure.

php

PHP (Hypertext Preprocessor) is a widely-used, open-source, server-side scripting language primarily designed for web development. It allows developers to create dynamic and interactive websites by embedding code directly within HTML. PHP is executed on the server, generating HTML that is then sent to the client's web browser. It's known for its ease of learning, compatibility with various databases, and robust ecosystem of frameworks and libraries.

github

GitHub is a web-based platform for version control and collaboration using Git. It's primarily used for software development, allowing developers to track changes to their code, collaborate on projects, and manage different versions. GitHub also provides features like issue tracking, pull requests, and project management tools, making it a central hub for many development workflows. Essentially, it's a place to store, share, and work on code collaboratively.
